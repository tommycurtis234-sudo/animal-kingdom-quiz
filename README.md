A fun, educational Progressive Web App quiz game about animals!

## Features
- 10 Animal Packs (Mammals, Birds, Reptiles, Fish, Amphibians, Insects, Big Cats, Sharks, Jungle, Arctic)
- 186 Quiz Questions with photos and facts
- Multiplayer mode (1-4 players)
- XP & Level progression system
- Daily challenges and streaks
- 20+ achievement badges
- Coin economy
- Text-to-speech support
- Works offline (PWA)
- 5 color themes
- Kids and Adults difficulty modes

## Getting Started
```bash
npm install
npm run dev
```

Built with React, TypeScript, Vite, and Tailwind CSS.
